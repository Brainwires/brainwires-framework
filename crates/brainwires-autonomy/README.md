# brainwires-autonomy

Autonomous agent operations for the Brainwires Framework — self-improvement, Git workflows, and human-out-of-loop execution.

## Features

- **Agent Operations** — attention management, health monitoring, hibernation, parallel execution, supervision
- **Git Workflow Automation** — branch lifecycle, PR management, merge policies, webhook handling
- **Self-Improvement** — feedback-driven strategy selection, code quality scanning, test coverage analysis
