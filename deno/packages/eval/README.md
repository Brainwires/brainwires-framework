# @brainwires/eval

Evaluation harness: trial runner (Wilson-score CI), evaluation cases, suite
orchestration, recorder, adversarial tests, regression suites, stability tests,
ranking metrics (NDCG / MRR / precision@k), fault analysis.

Extracted from `@brainwires/agents` in v0.11.0 to mirror Rust's
`brainwires-eval` crate.
